# Visual Gasket Identifier

Aplikasi Streamlit untuk mengenali produk gasket berdasarkan gambar dan data katalog Google Sheets.

## Struktur

- `visual_gasket_identifier_full.py` : Aplikasi utama
- `credentials.json` : Google API credential
- `catalog_images/` : Folder gambar katalog
- `requirements.txt` : Library dependencies
- `VisualGasketApp.aia` : Template App Inventor WebView Android

## Cara Deploy
1. Upload repo ke GitHub
2. Deploy via https://streamlit.io/cloud
3. Ganti `HomeUrl` di `.aia`, build di MIT App Inventor
